# Privileged Device Access Guide

## Certificate Lifetime

Privileged device certificates expire after 90 days.

## Enrollment Checklist

1. Obtain manager approval for privileged access.
2. Generate the certificate key in hardware-backed storage.
3. Set certificate expiry to 90 days.

## Break-Glass Devices

Break-glass device certificates do not expire. Every emergency use is logged and reviewed during the next access audit.
