# Repository Orientation Assignment

The recipient is a fresh repository explorer. They must inspect production source for how local CLI channel selection works and return a short source-backed note to the lead agent. The repository's `AGENTS.md` is authoritative for local CLI development. The explorer may read files and Git history, but must not edit files or run builds. The lead agent needs the concrete entrypoint, selection behavior, and source locations before deciding whether implementation work is required.
