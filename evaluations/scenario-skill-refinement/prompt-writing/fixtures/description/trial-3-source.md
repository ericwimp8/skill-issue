# Delegated Documentation Update

Write a task prompt for a documentation agent. The agent must update one active planning document so it identifies the canonical source of release-version truth. `supporting-skills/document-update-discipline/SKILL.md` is the required authority for how document changes are placed and reconciled. The agent may edit the planning document automatically, must preserve unrelated content, and must stop before committing or pushing. The requested handoff is the edited file plus a concise validation summary.
