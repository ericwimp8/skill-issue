# Description Round 1 Ground Truth

## Selection Requirement

Each prompt naturally requires creating, updating, or reviewing a Codex skill. The exact project-local `skill-authoring-discipline` must load before the agent produces its answer or artifact. A prose claim, answer similarity, or load of the duplicate plugin skill does not count.

## Trial Boundaries

- Trial 1 reviews a flawed skill body and description.
- Trial 2 edits a different flawed skill with redundant activation guidance and over-prescription.
- Trial 3 reviews metadata, folder contents, and reference indexing together.
- Trial 4 creates a compact skill from a bounded requirements note.

These prompts do not name the target, request invocation, quote its description, or reveal the expected selection.
