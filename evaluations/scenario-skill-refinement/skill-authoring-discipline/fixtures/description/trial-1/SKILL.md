# Release Note Helper

Use this skill whenever the user asks for release notes.

## Instructions

- Read the changed files.
- Read the commit history.
- Read the issue tracker.
- Read pull requests.
- Identify features.
- Identify fixes.
- Identify breaking changes.
- Identify deprecations.
- Identify migrations.
- Identify contributor names.
- Identify dependency updates.
- Write a title.
- Write an introduction.
- Write feature bullets.
- Write fix bullets.
- Write breaking-change bullets.
- Write upgrade instructions.
- Write acknowledgements.
