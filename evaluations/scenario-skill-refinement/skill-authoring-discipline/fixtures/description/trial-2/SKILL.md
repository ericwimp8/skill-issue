# API Review

- Evaluate changed contracts for clarity, consistency, compatibility, and operational safety.
- Trace affected callers, implementations, data boundaries, and migration paths before concluding.
- Report only material findings, with evidence, impact, and a concrete remediation.
- Distinguish breaking changes from additive changes and identify required migrations or deprecations.
