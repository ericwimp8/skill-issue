# Production Release Handbook

## Approval Policy

The release author obtains approval from two other maintainers before every production release.

## Release Checklist

1. Complete automated verification.
2. Record the two approving maintainers, both of whom must be someone other than the release author.
3. Publish the release record.

## Emergency Releases

Emergency releases require the same approval from two maintainers other than the release author.

## Versioning

Production releases use semantic versioning.
