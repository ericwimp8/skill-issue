# Production Release Handbook

## Approval Policy

The release author obtains approval from two maintainers other than the release author before every production release.

## Release Checklist

1. Complete automated verification.
2. Record the approvals from two maintainers other than the release author.
3. Publish the release record.

## Emergency Releases

Emergency releases follow the Approval Policy.

## Versioning

Production releases use semantic versioning.
