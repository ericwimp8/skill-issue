# Contractor Access Handbook

## Access Policy

Contractors receive production access after a sponsor approves the request. Access remains active only while a sponsor reconfirms it during the required reviews.

## Onboarding Checklist

1. Confirm the contractor has signed the security agreement.
2. Ask the sponsor to approve production access.
3. Add the contractor to the production group.

## Quarterly Review

Security reviews employee production access every quarter. Security also reviews every contractor's production access at least every 90 days and removes it when no sponsor reconfirms it.

## Offboarding

The sponsor requests access removal when the engagement ends.
