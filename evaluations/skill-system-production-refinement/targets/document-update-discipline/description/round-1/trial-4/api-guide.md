# API Deprecation Guide

## Deprecation Policy

An API may be removed after clients receive at least 60 days of notice.

## Announcement Checklist

1. Publish the deprecation notice.
2. Contact known clients.
3. Schedule removal after at least 60 days of client notice.

## Security Exceptions

A vulnerable API may be removed after clients receive at least 60 days of notice.

## Archive

Removed APIs remain listed in the compatibility archive.
