# Stop-State Actions

## Description Stop

- Action: Pause the current target's campaign after five unsuccessful description rounds, retain the evidence, and leave the state stopped until the user responds.
- Required question: **What specific number of additional description rounds do you authorize for this target?**

## Body Stop

- Action: Stop the entire campaign after five unsuccessful body cycles, report the retained failures, and do not continue this target or begin another target until the user responds.
- Required question: **What specific number of additional body cycles do you authorize for this target?**
