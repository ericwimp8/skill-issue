# Work Block 1 Supporting Plan: Research Cross-Harness Sub-Agent Launch

## Goal

Find out how sub-agents are launched in each selected harness.

## Research Task

Use one researcher for each of the nine selected harnesses:

- GitHub Copilot
- Claude Code
- OpenAI Codex
- Cursor
- Google Antigravity or Gemini CLI
- Grok Build
- OpenCode
- Kilo Code
- Pi

Each researcher answers only:

1. Does the harness support sub-agents?
2. If it does, how is a sub-agent launched?
3. Can a skill launch or request a sub-agent?

## Output

- One short report for each harness.
- One combined reference containing the nine answers.

## Completion

The research is complete when the combined reference states whether each harness supports sub-agents, how they are launched, and whether a skill can launch or request one.

## Result

Complete. The nine assignment reports and combined reference are stored under `plans/research/harness-subagent-launch/`.
