# Generation Ownership Map

- **Behavior owner:** `SKILL.md` owns the reusable investigation sequence, evidence boundary, causal classification, remediation direction, verification contract, and safe authority boundary.
- **Conditional platform detail:** none is required; the workflow must reason from supplied CI configuration rather than ship provider-specific reference material.
- **Deterministic repeated behavior:** none warrants a script; log formats and repository tooling vary, and contextual source tracing is the core work.
- **Output material:** no asset or template is required; concise report headings can live directly in the skill body.
- **Host metadata:** `agents/openai.yaml` owns only display metadata and must not duplicate behavior or add a default prompt.
- **Preserved meanings:** missing evidence stays explicit, source remains authoritative over tests, remote mutations require explicit authorization, and unrelated repository behavior remains outside remediation scope.
