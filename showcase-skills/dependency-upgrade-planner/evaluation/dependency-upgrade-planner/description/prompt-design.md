# Description Prompt Design

## Initial Pair

1. Plan the React 18.2 to stable React 19 upgrade for `fixtures/react-app/` before dependency changes.
2. Assess and sequence a Vite 5.4 to Vite 6 migration for `fixtures/vite-library/`, including build and deployment constraints.

## Confirmation Pair

3. Produce a reviewable Pydantic 1.10 to 2.x migration plan for `fixtures/pydantic-service/` without modifying it.
4. Map the dependency and platform work needed before the synthetic widget library can adopt Vite 6, emphasizing evidence gates and rollback.

Each prompt naturally requests dependency-upgrade planning without naming a skill, asking for invocation, quoting its description, or revealing the expected selection. Confirmation prompts differ in ecosystem and framing from the initial pair.
