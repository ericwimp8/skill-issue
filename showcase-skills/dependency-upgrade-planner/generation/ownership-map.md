# Generation Ownership Map

- **Behavior:** dependency-upgrade planning from project and upstream evidence.
- **Core owner:** `SKILL.md` owns the ordered discovery, research, impact, sequencing, evidence, validation, rollback, and output behavior.
- **Conditional platform detail:** owned by the inspected project's manifests and authoritative upstream sources at execution time; no packaged reference is justified.
- **Deterministic repeated behavior:** no stable cross-ecosystem computation warrants a script.
- **Output material:** no fixed asset is required because plan shape must follow the inspected dependency graph and repository conventions.
- **Host metadata:** `agents/openai.yaml` owns Codex interface copy only.
- **Preserved boundary:** the skill produces a plan and may run read-only inspection; it does not mutate dependency state.
